# 09 個股籌碼 K 線 / 分點足跡

> 這是產品的招牌功能,但也是最容易做過頭的功能。分三版收斂野心。

## V1 — 看得到足跡(夠用就好)

**畫面**(lightweight-charts):

```
┌───────────────────────────────┬──────────────┐
│ 日K + MA5/20/60 + 訊號標記 ▲  │ 分點面板      │
│ (240 日,可縮放)              │ 當日買超前15  │
│───────────────────────────────│ 當日賣超前15  │
│ 成交量柱                       │ (張/金額/佔比│
│───────────────────────────────│  連買天數/    │
│ 前15大買賣超淨額堆疊柱(綠紅) │  ⚠️隔日沖)   │
└───────────────────────────────┴──────────────┘
```

**互動**:
1. 點面板中任一分點 → K 線上疊該分點標記:買超日 ▲(綠,大小按金額 3 級)、賣超日 ▼(紅);同時下方柱圖切換為「該分點單獨淨買賣」。
2. 同時疊加上限 3 個分點(顏色區分),避免圖面爆炸。
3. 篩選器:只顯示關鍵分點/地緣分點/金額 ≥ X 萬/連買 ≥ 3 日。
4. 點 K 線上任一日 → 浮窗顯示該日前 5 買/賣分點速覽。
5. 分點歷史表(點分點後下方):日期、買賣超張/金額、收盤價、後 1/3/5/10/20 日報酬(色階);表尾統計列:出手次數、5 日勝率、平均報酬。

**V1 明確不做**:可信度分數、自動評語、勝率篩選、權證標記疊加、分時足跡。

**資料邊界(要在 UI 註明)**:僅有每日前 15 大買賣超與名單分點(見 05 §3),小額分點不完整;分點資料為 T+1。

## V2 — 看得懂足跡

1. 可信度分數與評語(04 §5)進面板:每個分點名旁顯示可信度徽章;點開看評語(「出手 14 次,5 日勝率 64%,偏波段,近期仍有效」)。
2. 新篩選:只顯示可信度 ≥ 70 / 只顯示非隔日沖。
3. K 線標記 hover 顯示「該次買進後 5 日報酬 +4.2%」。
4. 疊加層擴充:權證異動日(⚡底色)、入選觀察清單日(★)、盤中 A/B 訊號點位。
5. 分點視角頁:反向查詢——選一個分點,看它近 60 日在所有股票的進出與績效。
6. 布林通道、ATR 上圖。

## V3 — 用足跡做決策

1. 分點×題材勝率矩陣(「此分點在重電股勝率 71%,在其他僅 48%」)。
2. 多分點聯動偵測:經常同進同出的分點群(相關係數 + 同日進場頻率),標記為「疑似同一資金」。
3. 買後報酬分布圖(不只均值,給箱型圖)。
4. 參數化回測掛鉤:「只跟這個分點買,過去兩年績效曲線」。

## 實作備註

- 前端一次載入 240 日全量(四表合一 endpoint,見 05 §7),分點切換純前端運算,零額外請求 → 互動零延遲。
- 報酬計算基準:分點買超日的**次日開盤價**起算(分點資料 T+1 才看得到,用當日收盤自欺)。此原則全系統一致。
- 標記大小級距:佔當日成交值 1% / 3% / 6% 三級。
