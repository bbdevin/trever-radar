# 04 評分與訊號規則(全模型)

> 所有分數 0–100,風險為扣分制。所有門檻是「起始值」,V2 起用訊號績效追蹤數據調整。
> 所有規則必須是純函式:輸入 = 資料庫快照,輸出 = 分數 + 理由陣列,可 golden-file 測試、可回測重放。

## 0. 通用前置過濾(不進雷達的股票)

- 日均成交金額(20 日)< 3,000 萬 → 排除(流動性不足,訊號無意義且滑價大)。
- 全額交割股 → 排除。處置股 → 排除(理由:分盤交易,盤中邏輯失效);注意股不排除但扣分。
- 上市未滿 60 個交易日 → 排除(指標與基期不足)。

## 1. 權證異動分數(0–100,權重 20%)

### 異動定義(以「標的股」為單位彙總其全部權證)

| 指標 | 計算 | 給分 |
|---|---|---|
| W1 金額倍數 | 今日該股全部認購權證成交金額 ÷ 近 20 日均值 | ≥3 倍 +30;≥2 倍 +20;≥1.5 倍 +10 |
| W2 量倍數 | 同上以量計 | ≥3 倍 +15;≥2 倍 +10 |
| W3 標的未大漲 | 標的股今日漲幅 < 3% 且 5 日累漲 < 8% | +20(權證先動、股票未動 = 最有價值的異動) |
| W4 連續性 | 連 2 日 W1≥1.5 倍 +10;連 3 日 +15 | 上限 +15 |
| W5 購售比 | 認購金額 ÷(認購+認售)≥ 0.75 且總額 > 500 萬 | +10 |
| W6 集中度 | 單一權證佔該股權證總額 > 60%(疑似特定人) | +10 |

**品質過濾(不給分反而列風險)**:剩餘天期 < 45 天的權證金額不計入 W1/W2(末日權投機噪音);價外 > 20% 的深價外權證減半計;買賣價差 > 5% 的權證註記流動性差。

**基期門檻**:近 20 日權證日均金額 < 100 萬的標的,W1 倍數再高也封頂 40 分(低基期暴增常是單一散戶)。

### 你問的「主力提前卡位怎麼判斷」

無法直接證明,只能用組合間接推論:`W1≥2 + W3 + W4連續 + W5購售比高 + 標的分點分數≥60`。同時成立時,理由文字寫「權證連續放大且標的未漲,型態類似先行卡位」,並附風險「亦可能為發行商避險或散戶跟風」。誠實標註不確定性。

### 免費做不到的(明列)

權證分點成交明細(誰在買)→ 需逐權證查 bsr,V1/V2 不做;隱含波動率變化 → 需自算或付費,V3 選配。

## 2. 分點籌碼分數(0–100,權重 35%,全案最重)

### 每股每日先算的中間量

- `netbuy(branch, stock, day)`:分點淨買超張數與金額。
- 連買天數:netbuy > 0 且金額 ≥ 日成交值 0.5% 的連續天數。
- 集中度:前 5 大買超分點合計金額 ÷ 當日成交值(買方集中度);同理賣方。
- 大戶淨流:前 15 大買超合計 −前 15 大賣超合計。

### 給分

| 指標 | 條件 | 給分 |
|---|---|---|
| B1 連買 | 任一分點連買 ≥3 日且累計 ≥ 成交值 3% | +20;≥5 日 +30 |
| B2 多點同步 | ≥3 個分點同日買超各 ≥ 成交值 1% | +15 |
| B3 集中度躍升 | 買方集中度 ≥ 15% 且 > 20 日均值 ×1.5 | +15 |
| B4 地緣買超 | 地緣分點(§3)淨買 ≥ 成交值 1%,連 2 日再 +5 | +10~15 |
| B5 關鍵分點 | 關鍵分點(§4)淨買 ≥ 成交值 1% | +15;可信度 ≥70 的分點 +20 |
| B6 大戶淨流正 | 連 3 日為正 | +10 |
| B7 法人同向 | 投信買超 ≥1% 或外資連 3 買 | +10(法人項,併入此分) |
| 扣分 | 隔日沖名單分點佔買超前 3 名 | −15/家,最多 −30 |
| 扣分 | 昨日大買分點今日反手大賣(倒貨) | −20 |

### 你問的判斷方法

- **分點適合盤後分析嗎**:是,且只能盤後(T+1 資料)。盤中做不了分點追蹤,系統不假裝能。
- **隔日沖判定**(自動,滾動 60 日):某分點「買超日的次日即賣超 ≥ 買超量 70%」的比率 ≥ 60%,且樣本 ≥ 8 次 → 標記隔日沖。另人工維護知名隔日沖分點名單(V1 先人工,規則 V1.5 加上)。
- **買低賣高判定**:買超日收盤價位於其後 20 日價格區間的分位數,平均 < 0.4 → 常買低;賣超日分位 > 0.6 → 常賣高。樣本 ≥ 10 次才計。

## 3. 地緣券商模型

- 定義:分點所在縣市 = 公司總部或主要工廠所在縣市,且該分點非全國性大型分點(排除信義、館前這類什麼都買的)。
- V1 做法:分點地址(券商公開資料)+ 公司地址(MOPS)→ 縣市級自動對照,再人工覆核熱門股(前 300 檔)產生 `geo_branch_map` 表。小城市權重高於雙北(台北分點地緣訊號弱,竹科/台南/高雄訊號強)。
- 訊號意義:地緣買超 + 連買 → 疑似知情資金,理由文字明講「地緣分點(XX市)連 3 日買超」。

## 4. 關鍵分點模型

- V1:人工名單(市場公認主力分點 + 你自己觀察的),表 `key_branches`,可加註「擅長題材」。
- V2 起自動晉升/降級:分點可信度分數(§5)≥ 70 且近 90 日仍活躍 → 自動列關鍵;連續 60 日績效差 → 移出。人工名單保留但標記來源(manual/auto)。

## 5. 分點可信度分數(V2,0–100)

樣本定義:該分點在該股票的「進場事件」= 淨買超 ≥ 成交值 1% 的日子(連續日合併為一次事件,以首日計)。**事件數 < 10 不評分,顯示「樣本不足」**。

| 成分 | 權重 | 計算 |
|---|---|---|
| 勝率 | 30 | 進場後 5 日報酬 > 0 的比率,線性映射 40%→0 分、70%→滿分 |
| 報酬 | 25 | 進場後 5 日平均報酬,0%→0、+5%→滿分 |
| 買點品質 | 15 | 買低分位(§2)映射 |
| 持有型態 | 10 | 非隔日沖 +滿分;隔日沖 0 |
| 近效性 | 20 | 近 90 日事件的 5 日報酬均值 vs 全期,衰退則按比例扣 |

輸出評語模板:「此分點於本股近 2 年出手 14 次,5 日勝率 64%,平均報酬 +3.2%,偏波段操作,近 3 個月依然有效 → 值得追蹤」/「…次日倒貨率 78%,判定隔日沖 → 買超訊號參考價值低」。

跨股票泛化:同分點在其他股票勝率 < 45% 而本股 > 65% → 標註「僅在本股有效,疑似特定人於特定標的」——這反而是加分訊號。

## 6. 技術分數(0–100,權重 20%)

### V1 實作(日級)

| 指標 | 條件 | 給分 |
|---|---|---|
| T1 均線 | 站上 20 日線 +10;站上 60 日線 +10;5>10>20 多頭排列 +15 |
| T2 突破 | 收盤創 20 日新高 +15;帶量(量 ≥ 20日均量 1.5 倍)再 +10 |
| T3 底部整理 | 60 日振幅 < 25% 且今日收盤位於區間上緣 10% 內(箱型將突破)+15 |
| T4 量價 | 連 2 日量增價漲 +10 |
| T5 動能 | RSI14 於 50–70 +5(>80 不給,列追高風險);MACD 柱翻正 +5;KD 金叉於 50 下 +5 |

### 不做與晚做

- **不做**:W 底、頸線、旗形等型態辨識——主觀、假訊號多、實作貴。用 T2/T3 的可量化替代品。
- **V2**:布林通道(收盤突破上軌 + 帶量)、ATR(觀察價與失效價距離)、盤中分 K 轉強(§9)。
- **假訊號最多的指標**(給你的直答):單獨 KD 金叉、單獨 MACD 金叉、單獨 RSI 超賣反彈。全部只當「加分項」不當觸發主因;觸發主因永遠是「籌碼 + 量價突破」的組合。

### 與籌碼搭配原則

技術分單獨 ≥ 80 但分點分 < 40 → 不入清單(純技術突破無籌碼 = 容易一日行情)。分點分 ≥ 60 且技術分 ≥ 50 才是本系統的目標型態。

## 7. 題材分數(0–100,權重 10%)

- V1:人工維護 `themes` / `stock_themes`(26 個題材起步)。熱度 = 0.4×(題材成分股當日平均漲幅 z-score)+ 0.3×(上漲家數比)+ 0.3×(成分股合計成交金額 20 日倍數 z-score),映射 0–100。個股題材分 = 其所屬題材熱度最大值。
- 判定:**第一版人工維護,正確且必要**——自動題材分類(新聞 NLP)錯誤率高,先讓人工標籤把地基打好,V3 才用 AI 做「建議新增標籤」而非全自動。
- 與盤中結合(V2):監控池內同題材 ≥3 檔同時觸發 B 級以上 → 升級理由「題材群動」。

## 8. 風險扣分(0 至 −40,夾在綜合分之後)

| 風險 | 判定 | 扣分 | 顯示文字 |
|---|---|---|---|
| 短線過熱 | 5 日累漲 > 20% 或 10 日 > 35% | −15 | 「短線漲幅已大,追價風險高」 |
| 爆量長上影 | 量 > 20日均 2.5 倍且上影 ≥ 實體 2 倍 | −10 | 「爆量長上影,疑高檔換手/出貨」 |
| 開高走低 | 開盤 +3% 以上收黑 K | −8 | |
| 隔日沖進駐 | §2 扣分外,佔比 > 30% 額外標紅 | −(已計) | 「買超主力含隔日沖分點,隔日賣壓重」 |
| 注意股 | 官方公告 | −10 | |
| 融資過熱 | 融資增速 > 股價漲速且資使用率 > 60% | −8 | |
| 外資/投信連賣 | 連 5 日 | −8 | |
| 大盤風險 | 加權指數 < 60 日線且 20 日線下彎 | 全清單門檻 +5 分(提高入選標準)並全域提示 | |
| 當沖過熱 | 當沖佔比 > 60% | −8 | 「當沖比例過高,波動劇烈」 |

| 壓力區 | 收盤距上方 60 日高點 < 3% 且該區曾爆量 | −5 | 「接近前高壓力區」 |

## 9. 盤中訊號(V2)

### 9.1 大單門檻(依流動性分級,不用固定張數)

以「20 日日均成交金額 ADV」分級:

| ADV | 單筆大單門檻 | 5 分鐘累積門檻 |
|---|---|---|
| 3千萬–1億 | ≥ 300 萬 | ≥ 900 萬 |
| 1–5 億 | ≥ 500 萬 | ≥ 1,500 萬 |
| 5–20 億 | ≥ 1,000 萬 | ≥ 3,000 萬 |
| > 20 億 | ≥ 2,000 萬 | ≥ 6,000 萬 |

另一律要求:單筆 ≥ 該股近 20 日單筆成交金額的 99.5 百分位(自 V2 起用歷史 tick 統計校準,首月先用上表)。

### 9.2 主動買賣判定

成交價 ≥ 當下賣一價 → 主動買(外盤);≤ 買一價 → 主動賣。Fugle trades 頻道含內外盤資訊。5 分鐘主動買比 = 主動買金額 ÷(主動買+主動賣金額)。

### 9.3 量比與同時段比較

- 量比 = 今日開盤至今累積量 ÷ 近 20 日「同一時刻」平均累積量(預先為每股建 5 分鐘粒度的累積量曲線基準表,盤後更新)。
- 金額比同理。量比 ≥ 2.5 = 異常放大;≥ 4 = 強異常。

### 9.4 盤中技術條件

站上 VWAP(現價 > 當日 VWAP)、回測 VWAP 不破(觸及 VWAP ±0.3% 後 10 分鐘內收回其上)、突破昨高、突破 5 日高、5 分 K 連 2 根收高且量增。

### 9.5 訊號分級

先決條件:該股昨日盤後綜合分 ≥ 55(監控池本來就以此篩過)。

| 級別 | 條件 | 含義 |
|---|---|---|
| **A** | 盤後分 ≥ 70 且 [大單事件 ≥2 或 5分鐘累積達標且主動買比 ≥ 65%] 且 量比 ≥ 2.5 且 站上 VWAP 且 突破昨高 且 現漲幅 < 6% | 籌碼好+今天真的在發動,且未追高 |
| **B** | 盤後分 ≥ 60 且大單事件 ≥1 且 [量比 ≥ 2 或 突破昨高] | 初步發動,觀察 |
| **C** | 大單或量比達標但技術未確認(未站上 VWAP / 未過昨高) | 僅記錄,預設不通知 |
| **追高警示** | 符合 A/B 但現漲幅 ≥ 6% 或距今日低點 > 8% | 降級顯示「已發動勿追,等回測」 |

- 「等待回測」判定:追高警示後,若回測 VWAP 或今日開盤價不破且再現大單 → 補發 B 級「回測不破」訊號。
- 防抖:同股同級訊號 30 分鐘冷卻;開盤前 15 分鐘(09:00–09:15)只記錄不通知(雜訊最高時段);尾盤 13:15 後不通知。

### 9.6 假訊號抑制(通用)

1. 一切盤中訊號都乘上「盤後籌碼底分」——籌碼差的股票再大單也最多 C 級。
2. 大單必須伴隨價格行為(站上 VWAP/過昨高),純大單不觸發(可能是出貨對敲)。
3. 處置股、當沖比 > 60%、漲停鎖死(無法成交意義)不觸發。
4. 每日每股訊號上限 3 次,全池 A 級一天超過 15 檔 → 自動提高門檻並提示「今日全市場過熱,訊號稀釋」。

## 10. 盤後綜合分數與觀察清單

```
raw = 0.35×分點分 + 0.20×權證分 + 0.20×技術分 + 0.10×題材分 + 0.15×法人融資分
final = clamp(raw + 風險扣分總和, 0, 100)
```

入選:final ≥ 65,依 final 排序取前 30;同分以分點分優先。另附「權證異動特別榜」(權證分 ≥ 70 但總分未達者,單獨列出註明「僅權證異動,籌碼未確認」)。

觀察價/失效價:觀察價 = max(今日高點, 箱型上緣) × 1.005(突破確認);失效價 = min(5 日線, 今日低點)(V2 改用 1.5×ATR)。

## 11. 理由文字產生(規則模板,不用 LLM)

每個給分項觸發時同步 push 一條 reason 物件 `{code, text, value}`,模板含實際數字:

> 「分點【凱基-台南】連 4 日買超共 2,830 張,佔期間成交量 6.1%(歷史 5 日勝率 66%)」
> 「今日認購權證成交金額 4,120 萬,為 20 日均值 4.2 倍,標的今日僅漲 1.8%」
> 「10:12 出現主動買進大單 1,240 萬,5 分鐘累積 3,600 萬,主動買比 71%,股價站上 VWAP 並突破昨高 187.5」

風險提醒同機制。綜合卡片文字 = 取分數最高 3 條理由 + 全部風險,固定順序拼接。好處:零 token 成本、每字可追溯、回測可重放。

## 12. 回測方法(V2 簡易/V3 完整)

- V2:「前瞻記錄法」——清單與訊號產生當下即凍結存檔,批次回填 1/3/5/10/20 日報酬(以次日開盤價為進場基準,誠實計算,不用當日收盤自欺)。輸出:各級訊號勝率、平均報酬、各分項規則的邊際貢獻(有無該規則的清單差異)。
- V3:參數化重跑——權重與門檻做成 config,對 2–3 年歷史資料重算每日清單,比較不同參數組合;防過擬合:參數只允許粗粒度網格,樣本外驗證(前 2 年調參、最近 6 個月驗證)。
